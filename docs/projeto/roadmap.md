# Roadmap

O objetivo do **Snapport** é ser a solução mais eficiente e leve para portfólios baseados no GitHub. Nosso desenvolvimento é guiado por três pilares: **Simplicidade, Performance e Customização.**

### O que vem por aí

#### 1. Experiência do Desenvolvedor (DX)
- [ ] **CLI Tool**: Um comando para gerar arquivos de configuração básicos.
- [ ] **Múltiplos Temas**: Presets de CSS (Dark, Light, Minimalist) prontos para uso.

#### 2. Funcionalidades de UI
- [ ] **Suporte a Grid Customizado**: Propriedade para definir o número de colunas (1 a 4).
- [ ] **Internacionalização (i18n)**: Traduções nativas para os botões de ação e busca.

#### 3. Core & Ecossistema
- [ ] **Plugin React/Vue**: Wrappers oficiais para facilitar o uso em frameworks modernos.
- [ ] **Image Proxy**: Opção para otimizar as imagens de preview via CDN.

> **Tem uma sugestão técnica?** Adoraríamos ouvir sua ideia. [Abra uma Feature Request](https://github.com/guilhermegodoydev/snapport/issues).
