Como Contribuir

On this page

Contribuindo com a Snapport 🤝 ​

Primeiramente, obrigado por considerar contribuir! Projetos de código aberto são movidos por pessoas como você. Não importa o seu nível de experiência, sua ajuda é valiosa.

Formas de Colaborar ​

Não é só com código que se constrói uma lib. Você pode ajudar de várias formas:

  • Expanda o Ecossistema: Adicione novas tecnologias ao nosso TECH_MAP. É a forma mais fácil e impactante de começar!
  • Feedback Visual: Testou a lib no seu site? Mande o link! Adoraríamos ver como você a personalizou.
  • Refine o Código: Se viu algo que pode ser otimizado ou um bug escondido, um Pull Request é sempre bem-vindo.

Configurando seu Ambiente ​

Para garantir que você consiga testar suas mudanças com segurança, preparamos um Playground integrado:

  1. Fork & Clone: Faça o fork do projeto e clone-o localmente.
  2. Instalação: Rode npm install.
  3. Desenvolvimento: Use npm run dev. Isso abrirá um ambiente de teste que consome o código da pasta src em tempo real.
  4. Verificação: Antes de enviar, certifique-se de que o build está passando com npm run build.

Por que contribuir? ​

Ao contribuir para a Snapport, você ganha visibilidade no seu perfil do GitHub, aprende mais sobre o ecossistema de bibliotecas e ajuda a construir uma ferramenta que outros desenvolvedores usarão em seus próprios portfólios.

Dúvidas? Sinta-se à vontade para abrir uma Issue com a tag question.
